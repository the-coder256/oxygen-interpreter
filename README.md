# Oxygen Intepreter v0.6
This program interprets Oxygen programs (.ox)

## How to Use
(assuming you are in the repo folder)
```
cd src
py main.py "<file_path>"
```

For example:
```
cd src
py main.py "../tests/test.ox"
```
